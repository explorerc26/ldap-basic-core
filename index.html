# Dummyldap by intelie

Dummyldap

Portable dummy LDAP server for development

View on GitHub

This project is a little hack.

Don't expect much more than a simple LDAP server to speed up your development.

Installation

mkdir dummyldap && cd $_
curl -Ls intelie.github.io/dummyldap | sh
To install another version: 
mkdir dummyldap && cd $_
curl -Ls intelie.github.io/dummyldap | sh -s 0.1
